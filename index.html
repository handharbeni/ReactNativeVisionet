Data Manipulation 1

Pada kesempatan kali ini Anda akan mempelajari bagaimana memanipulasi data, mengubah dari sebuah bentuk data menjadi bentuk yang lain, pada javascript.

Format Masukan

Anda akan mengerjakan sebuah fungsi yang memiliki 2 buah parameter.

  • Argumen pertama adalah data yang berbentuk sebuah object.
  • Argumen kedua adalah searchText yang berbentuk sebuah string.

Format Keluaran

Pada setiap test case, Anda harus menenentukan object-object mana sajakah yang termasuk dari kata kunci yang digunakan untuk mencari. Anda harus memperhatikan, search text dapat berupa nama, ataupun kategori, object harus di masukkan jika salah satu huruf saja termasuk pada nama ataupun kategori object tersebut, dan case insensitive, jika pencarian kosong, maka tampilkan semua data. Output berupa array object hasil dari pencarian tersebut.

Contoh Masukan 


  var data = {
    "Books": [{
      "id": 1,
      "name": "Gramedia",
      "floor": "G"
    },{
      "id": 2,
      "name": "Togamas",
      "floor": "UG"
    }],
    "Food & Beverages": [{
      "id": 3,
      "name": "KFC",
      "floor": "LG"
    }];
    var searchText = 'Gra';

Contoh Keluaran 


    [{
      "id": 1,
      "name": "Gramedia",
      "floor": "G",
      "category": "Books"
    }]

Penjelasan

Pada data hanya terdapat 1 object yang mengandung 'Gra' yaitu object dengan nama "Gramedia", sehingga object dengan name Gramedia tersebut dimasukkan ke dalam array, dan di jadikan sebagai return value functionnya

Untuk mengerjakan silakan buka index.js, dan kerjakan pada function search(data, searchText)
